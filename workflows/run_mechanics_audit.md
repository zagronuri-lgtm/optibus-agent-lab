# Run Mechanics Audit

## Purpose

Inspect run-mechanics dependencies before the Run Readiness Gate.

## Required checks

- Algorithm Parameters
- Algorithm choice justification
- Advanced Vehicle Adapter warning acceptance when selected
- DEEP configuration when required
- Trip Connections
- Vehicle Piece Validation readiness
- Deadhead Catalog
- Validation Panel issues
- Pull reliefs when needed

## Exit criteria

- Algorithm choice is justified.
- DEEP requirements are configured or marked as blockers.
- Vehicle Piece Validation readiness is recorded.
- Validation issues are classified before readiness review.
