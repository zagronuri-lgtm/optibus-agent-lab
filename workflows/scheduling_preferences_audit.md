# Scheduling Preferences Audit

## Purpose

Inspect Optibus scheduling preferences that materially affect run behavior.

## Required checks

- Cost preferences
- Depot Setup
- Midday Park
- Pre/Post Trip
- Relief Points
- Relief Timing
- Duty Types
- Work Limitation
- Time Limitations
- Split Break Definition
- Limit Short Pieces
- Crew Relaxation
- Global Constraints

## Exit criteria

- Every preference is classified as validated, missing, not applicable, or unknown.
- Hard and soft constraints are identified.
- Unknown or incomplete Duty Types block Run readiness.
