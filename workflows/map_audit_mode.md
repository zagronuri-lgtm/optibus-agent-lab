# Map Audit Mode

## Purpose

Collect read-only baseline evidence for the target Optibus map.

## Required evidence

- Map name and scenario/version identifiers.
- Copy/snapshot status.
- Baseline KPIs required by the scenario.
- Visible warnings and validation panel status.
- Screenshots before and after audit.

## Exit criteria

- Baseline KPIs are collected or explicitly marked missing.
- No copy/snapshot is recorded as a Run blocker.
- Findings are separated into facts, assumptions, risks, and recommendations.
