# Knowledge Base

The Optibus Agent Lab treats `knowledge/optibus_mastery.md` as its authoritative internal operating manual.

The uploaded `optibus_mastery.md` file was not present in this workspace when this scaffold was updated, so the repository includes a structured knowledge-base file at that path containing the required run-readiness and failure-diagnosis rules from the task. Replace or extend `knowledge/optibus_mastery.md` with the full internal source whenever it is available.

Framework code loads the knowledge file in plan and browser modes and includes its path in generated logs/reports. Browser automation must follow the knowledge base before applying any workflow-specific config.
